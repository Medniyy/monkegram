# Local fonts

MonkeGram self-hosts the same Google Fonts used by the tested app so production
builds do not depend on downloading fonts during `next build`.

- Press Start 2P: https://fonts.google.com/specimen/Press+Start+2P
- VT323: https://fonts.google.com/specimen/VT323

Both are distributed under the SIL Open Font License 1.1. Their license files
are stored alongside the font binaries.
